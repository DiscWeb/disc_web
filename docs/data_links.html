Documentation by Filetype

The Chronic Conditions Warehouse (CCW) keeps a list of codebooks for Medicare and Medicaid data as well as record layouts for other less common files. It also publishes documentation for other files it creates, specifically the chronic conditions flags for Medicare, and some other data products.

Primary (27 or 30) Chronic Conditions: This Chronic Conditions link contains the most recent chronic conditions algorithms. Notably, these codes are updated yearly based on an expert panel, but flags from prior years are not rerun on the latest algorithms -- CCW usually recalculates flags for the two years prior to the year of the algorithm change.

Other (40) Chronic Health, Mental Health, and Potentially Disabling Conditions: These "Other" conditions contain codes for mental health disorders, substance use, and other conditions. This conditions file was created in 2017 and has had updates in the years since. For a flag like OUD#4 (MOUD), the coding of naltrexone depends on the flag for substance use disorder (SUD) and alcohol use disorder (AUD), so replicating that also requires replicating those flags.

CCW User Documentation These User Data Guides from CCW describe how to work with Medicare claims data as well as some of the other data sets. This link also includes guides on the VRDC export review process.

Public RESDAC Documentation

RESDAC publishes other documentation that describes how to use TAF claims data on the VRDC, including information on how to link header and line files or link beneficiaries in across years.

TAF Data Quality Resources :

  • DQ-Atlas: This website is run by the government to provide data about TAF data quality by state and year across many different variables in the data.
  • Medicaid Demographic File: The Annual DE File includes demographic and eligibility data for all Medicaid and CHIP beneficiaries who were enrolled for at least one day during the calendar year. It provides monthly enrollment information, eligibility categories, and benefit package details.
  • Claims Files: This documentation describes the inpatient, long-term care, pharmacy, and other medical services data. Each claim includes a header file, summarizing the entire claim, and line files detailing each service within the claim, which are linked for comprehensive analysis.
  • Annual Managed Care Plan Files: This documentation describes managed care entitys that are authorized to enroll Medicaid or CHIP beneficiaries. The variables include the planâ€™s business characteristics (such as profit status), its contract with the state (such as type of reimbursement arrangement and operating authority), and the location and type of beneficiaries the plan is allowed to enroll.
  • Annual Provider Files: The provider (APR) files capture information about all providers authorized by a state to render services to Medicaid and CHIP beneficiaries at any point in the calendar year, as well as various providers with pending or denied authorization.
  • Illinois Claims : Illinois TAF data are based on a system of claims adjustments (instead of finalized claims) and so may require extra steps for researchers to analyze.
